### Installation

- Just, Clone this repository -
```
$ git clone git://github.com/Akshay-Arjun/69phisher.git
```

- Change to cloned directory and run `69phisher.sh` -
```
$ cd 69phisher
$ chmod 777 69phisher.sh
$ bash 69phisher.sh
```

**`69phisher`** requires following programs to run properly -
